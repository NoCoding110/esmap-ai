   ```bash
   cd esmap-ai
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

   ```yaml

   Node.js version: 18.x
   Environment variables: (none required)
   ```
